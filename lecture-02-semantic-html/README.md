## 2. Semantic decisions I made (REQUIRED)

### Decision 1
- Element(s) used: `<header>`
- Where in the page: At the top of the document
- Why this element is semantically correct:
  The header contains the main title of the page and personal introductory information, which defines the identity and purpose of the page.

### Decision 2
- Element(s) used: `<section>`
- Where in the page: Portfolio and Areas of Interest sections
- Why this element is semantically correct:
  Each section represents a standalone thematic grouping of related content, making the structure clear for both users and screen readers.

### Decision 3
- Element(s) used: `<figure>` and `<figcaption>`
- Where in the page: Around images and embedded video in the portfolio
- Why this element is semantically correct:
  These elements associate media with descriptive captions, improving meaning and accessibility.
